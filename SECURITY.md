# Security Policy

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in Nexus Proxy, please do not report it via a public GitHub issue. Instead, please report it privately using the [GitHub Security Advisories](https://github.com/anchapin/nexus-proxy/security-advisories/new) feature.

We will acknowledge your report within 48 hours and work to resolve the issue before public disclosure.

## Supported Versions

We provide security updates for the most recent minor release of Nexus Proxy. If you are using an older version, please upgrade to the latest release to ensure you have the most recent security patches.

Backports for critical security fixes to older versions may be provided on a case-by-case basis.
